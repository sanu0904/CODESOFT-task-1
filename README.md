📌 CODESOFT-task-1

A basic chatbot created for learning purposes and internship projects. It can respond to simple user messages like greetings, the chatbot's name, the developer's name, and basic questions.

It uses predefined rule-based responses to match user input and reply accordingly. The project is simple to understand and is a great starting point for learning how rule-based conversation systems work.

✨ **Features**

* Rule-based responses
* Interactive conversation
* Exit option using "bye"

🛠️ **Technologies Used**

* Python
